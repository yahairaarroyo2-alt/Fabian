# docs/

Esta carpeta contiene documentación y versiones alternativas del proyecto FBA Fit.

## Archivos

### `workout-standalone.html`

Versión completamente autocontenida de la app de rutinas, **sin dependencias de React ni Node.js**.

- Se puede abrir directamente en cualquier navegador como archivo local
- No requiere `npm install` ni servidor de desarrollo
- Útil para compartir la rutina con usuarios que no tienen el entorno de desarrollo configurado
- Contiene los datos de la rutina, estilos y lógica en un solo archivo HTML

**Cómo usarla:** Abrir el archivo directamente en el navegador, o servir desde cualquier servidor HTTP estático.
